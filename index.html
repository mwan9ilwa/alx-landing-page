About This Project

This project was inspired by the need for developers to easily track their GitHub statistics and repository information in a centralized platform. The idea stemmed from personal experiences of wanting to monitor my GitHub activity more effectively and share it with potential employers or collaborators.

As a portfolio project for Holberton School, this application demonstrates the skills and technologies I have learned, including HTML, CSS, JavaScript, and Bootstrap.
