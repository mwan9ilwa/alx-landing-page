Key Features

Contributions Chart

The Contributions Chart, often referred to as the "contributions calendar" or "activity graph," visualizes a user's GitHub activity over the past month. It highlights the number of contributions (commits, pull requests, issues, etc.).

Top Languages

The Top Languages feature displays a breakdown of the programming languages a user has used across their public repositories. This is often shown as a pie chart, representing the percentage of code written in each language.

Recent Activities

The Recent Activities section lists the latest actions a user has taken on GitHub. This includes activities such as commits, pull requests, issues, comments, and stars. It provides a chronological log of recent contributions and interactions.

About This Project

This project was inspired by the need for developers to easily track their GitHub statistics and repository information in a centralized platform. The idea stemmed from personal experiences of wanting to monitor my GitHub activity more effectively and share it with potential employers or collaborators.

As a portfolio project for Holberton School, this application demonstrates the skills and technologies I have learned, including HTML, CSS, JavaScript, and Bootstrap.
